Pagefind Hub Example

A comprehensive showcase for aggregating content from diverse sources and enabling high-performance search with Pagefind.

Default UI

The standard Pagefind search interface, providing a complete search experience in a single UI component.

Component UI

Modular search elements as Web Components, allowing for flexible layouts and deeper customization.

Configuration Overview

This showcase indexes data from various sources, including:

  • Local Files: Static HTML content and documentation
  • Social Media: Recent posts and updates from social platforms
  • Video Platforms: Video metadata and channel information
  • External APIs: Data fetched from custom providers and APIs

Check example/pagefind-hub.config.ts for the full configuration.
